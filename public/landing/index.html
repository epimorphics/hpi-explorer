Open Data

As part of our commitment to the Government’s priorities of economic growth and data transparency Land Registry publishes the following public datasets:

  • House Price Index background data – available from January 1995 and updated in full each month.
  • Price paid data – available from January 1995 and updated in full each month.
  • Transaction data – available from December 2011.

The above datasets are provided in comma-separated value (csv) and linked data formats, with Price Paid Data also available as a text file. These datasets can be downloaded in full by accessing our Public Data. For more information on our linked data, please see below.

For our Price Paid and House Price Index customers that do not require the full dataset but are interested in running bespoke reports we have developed the following options:

Search the House Price Index

Use this search to find average price, sales volumes and property type for a specified location over a defined period of time. Our House Price Index compares average house price today to what it was in January 1995, with the index set then at 100. It includes figures at national, regional, county and Local Authority level. For more information visit About the House Price Index.

To better understand the data model, or learn more about using the data, please see the detailed documentation.

Price Paid Data Report Builder

Use this search to build bespoke reports using our detailed price paid data. Reports can be based on location, tenure, price paid or property type over a defined period of time.

Our price paid data tracks the residential property sales in England and Wales that are lodged with us for registration. The dataset is a reliable source of house price information and consists of more than 18.5 million definitive records dating back to January 1995. For more information on this dataset and what it does and doesn't include, see the Price Paid Data FAQs.

We strive to ensure that our public data is as accurate as possible but cannot guarantee that it is free from errors or fit for your purpose or use. Reports are based on data collected at the time a property transaction is registered with us and will not necessarily be up to date with the most recent information. See this FAQ entry for more information.

To better understand the data model, or learn more about using the data, please see the detailed documentation.

Linked Data

We also publish this data in linked data format.

If you would like to try out your own SPARQL queries against our linked data publications, please use this SPARQL form. We have included some example queries with the form, and you can generate the output as a table, in plain text, as a JSON file or in XML.

If you want to see the SPARQL query that is used to generate the reports in the HPI and PPD reporting tools described above, you can do this by selecting the 'view SPARQL query' actions from the 'download data' view in each search tool. You can even amend and re-run the SPARQL query as required.
